иЎЁж јжқҘжәҗ:yirlinзҡ„еҚҡе®ў

  • \: е°ҶдёӢдёҖдёӘеӯ—з¬Ұж Үи®°дёәдёҖдёӘзү№ж®Ҡеӯ—з¬ҰгҖҒжҲ–дёҖдёӘеҺҹд№үеӯ—з¬ҰгҖҒжҲ–дёҖдёӘ еҗ‘еҗҺеј•з”ЁгҖҒжҲ–дёҖдёӘе…«иҝӣеҲ¶иҪ¬д№үз¬ҰгҖӮдҫӢеҰӮпјҢn еҢ№й…Қеӯ—з¬Ұ nгҖӮ\n еҢ№й…ҚдёҖдёӘжҚўиЎҢз¬ҰгҖӮеәҸеҲ— \\ еҢ№й…Қ \ иҖҢ \( еҲҷеҢ№й…Қ (гҖӮ
  • ^: еҢ№й…Қиҫ“е…Ҙеӯ—з¬ҰдёІзҡ„ејҖе§ӢдҪҚзҪ®гҖӮеҰӮжһңи®ҫзҪ®дәҶ RegExp еҜ№иұЎзҡ„ Multiline еұһжҖ§пјҢ^ д№ҹеҢ№й…Қ \n жҲ– \r д№ӢеҗҺзҡ„дҪҚзҪ®гҖӮ
  • \$: еҢ№й…Қиҫ“е…Ҙеӯ—з¬ҰдёІзҡ„з»“жқҹдҪҚзҪ®гҖӮеҰӮжһңи®ҫзҪ®дәҶRegExp еҜ№иұЎзҡ„ Multiline еұһжҖ§пјҢ\$ д№ҹеҢ№й…Қ \n жҲ– \r д№ӢеүҚзҡ„дҪҚзҪ®гҖӮ
  • *: еҢ№й…ҚеүҚйқўзҡ„еӯҗиЎЁиҫҫејҸйӣ¶ж¬ЎжҲ–еӨҡж¬ЎгҖӮдҫӢеҰӮпјҢzo* иғҪеҢ№й…Қ z д»ҘеҸҠ zooгҖӮ* зӯүд»·дәҺ{0,}гҖӮ
  • +: еҢ№й…ҚеүҚйқўзҡ„еӯҗиЎЁиҫҫејҸдёҖж¬ЎжҲ–еӨҡж¬ЎгҖӮдҫӢеҰӮпјҢzo+ иғҪеҢ№й…Қ zo д»ҘеҸҠ zooпјҢдҪҶдёҚиғҪеҢ№й…Қ zгҖӮ+ зӯүд»·дәҺ {1,}гҖӮ
  • ?: еҢ№й…ҚеүҚйқўзҡ„еӯҗиЎЁиҫҫејҸйӣ¶ж¬ЎжҲ–дёҖж¬ЎгҖӮдҫӢеҰӮпјҢdo(es)? еҸҜд»ҘеҢ№й…Қ do жҲ– does дёӯзҡ„do гҖӮ? зӯүд»·дәҺ {0,1}гҖӮ
  • {n}: n жҳҜдёҖдёӘйқһиҙҹж•ҙж•°гҖӮеҢ№й…ҚзЎ®е®ҡзҡ„ n ж¬ЎгҖӮдҫӢеҰӮпјҢo{2} дёҚиғҪеҢ№й…Қ Bob дёӯзҡ„ oпјҢдҪҶжҳҜиғҪеҢ№й…Қ food дёӯзҡ„дёӨдёӘ oгҖӮ
  • {n,}: n жҳҜдёҖдёӘйқһиҙҹж•ҙж•°гҖӮиҮіе°‘еҢ№й…Қn ж¬ЎгҖӮдҫӢеҰӮпјҢo{2,} дёҚиғҪеҢ№й…Қ Bob дёӯзҡ„ oпјҢдҪҶиғҪеҢ№й…Қ foooood дёӯзҡ„жүҖжңү oгҖӮo{1,} зӯүд»·дәҺ o+гҖӮo{0,} еҲҷзӯүд»·дәҺ o*гҖӮ
  • {n,m}: m е’Ң n еқҮдёәйқһиҙҹж•ҙж•°пјҢе…¶дёӯn <= mгҖӮжңҖе°‘еҢ№й…Қ n ж¬Ўдё”жңҖеӨҡеҢ№й…Қ m ж¬ЎгҖӮдҫӢеҰӮпјҢo{1,3} е°ҶеҢ№й…Қ fooooood дёӯзҡ„еүҚдёүдёӘ oгҖӮo{0,1} зӯүд»·дәҺ o?гҖӮиҜ·жіЁж„ҸеңЁйҖ—еҸ·е’ҢдёӨдёӘж•°д№Ӣй—ҙдёҚиғҪжңүз©әж јгҖӮ
  • ?: еҪ“иҜҘеӯ—з¬Ұзҙ§и·ҹеңЁд»»дҪ•дёҖдёӘе…¶д»–йҷҗеҲ¶з¬Ұ (*, +, ?, {n}, {n,}, {n,m}) еҗҺйқўж—¶пјҢеҢ№й…ҚжЁЎејҸжҳҜйқһиҙӘе©Әзҡ„гҖӮйқһиҙӘе©ӘжЁЎејҸе°ҪеҸҜиғҪе°‘зҡ„еҢ№й…ҚжүҖжҗңзҙўзҡ„еӯ—з¬ҰдёІпјҢиҖҢй»ҳи®Өзҡ„иҙӘе©ӘжЁЎејҸеҲҷе°ҪеҸҜиғҪеӨҡзҡ„еҢ№й…ҚжүҖжҗңзҙўзҡ„еӯ—з¬ҰдёІгҖӮдҫӢеҰӮпјҢеҜ№дәҺеӯ—з¬ҰдёІ ooooпјҢo+? е°ҶеҢ№й…ҚеҚ•дёӘ oпјҢиҖҢ o+ е°ҶеҢ№й…ҚжүҖжңү oгҖӮ
  • .: еҢ№й…ҚйҷӨ \n д№ӢеӨ–зҡ„д»»дҪ•еҚ•дёӘеӯ—з¬ҰгҖӮиҰҒеҢ№й…ҚеҢ…жӢ¬ \n еңЁеҶ…зҡ„д»»дҪ•еӯ—з¬ҰпјҢиҜ·дҪҝз”ЁиұЎ [.\n] зҡ„жЁЎејҸгҖӮ
  • (pattern): еҢ№й…Қ pattern е№¶иҺ·еҸ–иҝҷдёҖеҢ№й…ҚгҖӮжүҖиҺ·еҸ–зҡ„еҢ№й…ҚеҸҜд»Ҙд»Һдә§з”ҹзҡ„ Matches йӣҶеҗҲеҫ—еҲ°пјҢеңЁVBScript дёӯдҪҝз”Ё SubMatches йӣҶеҗҲпјҢеңЁJScript дёӯеҲҷдҪҝз”Ё $0вҖҰ$9 еұһжҖ§гҖӮиҰҒеҢ№й…ҚеңҶжӢ¬еҸ·еӯ—з¬ҰпјҢиҜ·дҪҝз”Ё \( жҲ– \)гҖӮ
  • (?:pattern): еҢ№й…Қ pattern дҪҶдёҚиҺ·еҸ–еҢ№й…Қз»“жһңпјҢд№ҹе°ұжҳҜиҜҙиҝҷжҳҜдёҖдёӘйқһиҺ·еҸ–еҢ№й…ҚпјҢдёҚиҝӣиЎҢеӯҳеӮЁдҫӣд»ҘеҗҺдҪҝз”ЁгҖӮиҝҷеңЁдҪҝз”ЁвҖқжҲ–вҖқеӯ—з¬ҰжқҘз»„еҗҲдёҖдёӘжЁЎејҸзҡ„еҗ„дёӘйғЁеҲҶжҳҜеҫҲжңүз”ЁгҖӮдҫӢеҰӮпјҢ industr(?:y|ies) е°ұжҳҜдёҖдёӘжҜ” industry|industries жӣҙз®Җз•Ҙзҡ„иЎЁиҫҫејҸгҖӮ
  • (?=pattern): жӯЈеҗ‘йў„жҹҘпјҢеңЁд»»дҪ•еҢ№й…Қ pattern зҡ„еӯ—з¬ҰдёІејҖе§ӢеӨ„еҢ№й…ҚжҹҘжүҫеӯ—з¬ҰдёІгҖӮиҝҷжҳҜдёҖдёӘйқһиҺ·еҸ–еҢ№й…ҚпјҢд№ҹе°ұжҳҜиҜҙпјҢиҜҘеҢ№й…ҚдёҚйңҖиҰҒиҺ·еҸ–дҫӣд»ҘеҗҺдҪҝз”ЁгҖӮдҫӢеҰӮпјҢWindows (?=95|98|NT|2000) иғҪеҢ№й…Қ Windows 2000 дёӯзҡ„ Windows пјҢдҪҶдёҚиғҪеҢ№й…Қ Windows 3.1 дёӯзҡ„ WindowsгҖӮйў„жҹҘдёҚж¶ҲиҖ—еӯ—з¬ҰпјҢд№ҹе°ұжҳҜиҜҙпјҢеңЁдёҖдёӘеҢ№й…ҚеҸ‘з”ҹеҗҺпјҢеңЁжңҖеҗҺдёҖж¬ЎеҢ№й…Қд№ӢеҗҺз«ӢеҚіејҖе§ӢдёӢдёҖж¬ЎеҢ№й…Қзҡ„жҗңзҙўпјҢиҖҢдёҚжҳҜд»ҺеҢ…еҗ«йў„жҹҘзҡ„еӯ—з¬Ұд№ӢеҗҺејҖе§ӢгҖӮ
  • (?!pattern)иҙҹеҗ‘йў„жҹҘпјҢеңЁд»»дҪ•дёҚеҢ№й…Қ pattern зҡ„еӯ—з¬ҰдёІејҖе§ӢеӨ„еҢ№й…ҚжҹҘжүҫеӯ—з¬ҰдёІгҖӮиҝҷжҳҜдёҖдёӘйқһиҺ·еҸ–еҢ№й…ҚпјҢд№ҹе°ұжҳҜиҜҙпјҢиҜҘеҢ№й…ҚдёҚйңҖиҰҒиҺ·еҸ–дҫӣд»ҘеҗҺдҪҝз”ЁгҖӮдҫӢеҰӮWindows (?!95|98|NT|2000) иғҪеҢ№й…Қ Windows 3.1 дёӯзҡ„ WindowsпјҢдҪҶдёҚиғҪеҢ№й…Қ Windows 2000 дёӯзҡ„ WindowsгҖӮйў„жҹҘдёҚж¶ҲиҖ—еӯ—з¬ҰпјҢд№ҹе°ұжҳҜиҜҙпјҢеңЁдёҖдёӘеҢ№й…ҚеҸ‘з”ҹеҗҺпјҢеңЁжңҖеҗҺдёҖж¬ЎеҢ№й…Қд№ӢеҗҺз«ӢеҚіејҖе§ӢдёӢдёҖж¬ЎеҢ№й…Қзҡ„жҗңзҙўпјҢиҖҢдёҚжҳҜд»ҺеҢ…еҗ«йў„жҹҘзҡ„еӯ—з¬Ұд№ӢеҗҺејҖе§Ӣ
  • x|y: еҢ№й…Қ x жҲ– yгҖӮдҫӢеҰӮпјҢz|food иғҪеҢ№й…Қ z жҲ– foodгҖӮ(z|f)ood еҲҷеҢ№й…Қ zood жҲ– foodгҖӮ
  • [xyz]: еӯ—з¬ҰйӣҶеҗҲгҖӮеҢ№й…ҚжүҖеҢ…еҗ«зҡ„д»»ж„ҸдёҖдёӘеӯ—з¬ҰгҖӮдҫӢеҰӮпјҢ [abc] еҸҜд»ҘеҢ№й…Қ plain дёӯзҡ„ aгҖӮ
  • [^xyz]: иҙҹеҖјеӯ—з¬ҰйӣҶеҗҲгҖӮеҢ№й…ҚжңӘеҢ…еҗ«зҡ„д»»ж„Ҹеӯ—з¬ҰгҖӮдҫӢеҰӮпјҢ [^abc] еҸҜд»ҘеҢ№й…Қ plain дёӯзҡ„pгҖӮ
  • [a-z]: еӯ—з¬ҰиҢғеӣҙгҖӮеҢ№й…ҚжҢҮе®ҡиҢғеӣҙеҶ…зҡ„д»»ж„Ҹеӯ—з¬ҰгҖӮдҫӢеҰӮпјҢ[a-z] еҸҜд»ҘеҢ№й…Қ a еҲ° z иҢғеӣҙеҶ…зҡ„д»»ж„Ҹе°ҸеҶҷеӯ—жҜҚеӯ—з¬ҰгҖӮ
  • [^a-z]: иҙҹеҖјеӯ—з¬ҰиҢғеӣҙгҖӮеҢ№й…Қд»»дҪ•дёҚеңЁжҢҮе®ҡиҢғеӣҙеҶ…зҡ„д»»ж„Ҹеӯ—з¬ҰгҖӮдҫӢеҰӮпјҢ[^a-z] еҸҜд»ҘеҢ№й…Қд»»дҪ•дёҚеңЁ a еҲ° z иҢғеӣҙеҶ…зҡ„д»»ж„Ҹеӯ—з¬ҰгҖӮ
  • \b: еҢ№й…ҚдёҖдёӘеҚ•иҜҚиҫ№з•ҢпјҢд№ҹе°ұжҳҜжҢҮеҚ•иҜҚе’Ңз©әж јй—ҙзҡ„дҪҚзҪ®гҖӮдҫӢеҰӮпјҢ er\bеҸҜд»ҘеҢ№й…Қnever дёӯзҡ„ erпјҢдҪҶдёҚиғҪеҢ№й…Қ verb дёӯзҡ„ erгҖӮ
  • \B: еҢ№й…ҚйқһеҚ•иҜҚиҫ№з•ҢгҖӮer\B иғҪеҢ№й…Қ verb дёӯзҡ„ erпјҢдҪҶдёҚиғҪеҢ№й…Қ never дёӯзҡ„ erгҖӮ
  • \cx: еҢ№й…Қз”ұ x жҢҮжҳҺзҡ„жҺ§еҲ¶еӯ—з¬ҰгҖӮдҫӢеҰӮпјҢ \cM еҢ№й…ҚдёҖдёӘ Control-M жҲ–еӣһиҪҰз¬ҰгҖӮx зҡ„еҖјеҝ…йЎ»дёә A-Z жҲ– a-z д№ӢдёҖгҖӮеҗҰеҲҷпјҢе°Ҷ c и§ҶдёәдёҖдёӘеҺҹд№үзҡ„ c еӯ—з¬ҰгҖӮ
  • \d: еҢ№й…ҚдёҖдёӘж•°еӯ—еӯ—з¬ҰгҖӮзӯүд»·дәҺ [0-9]гҖӮ
  • \D: еҢ№й…ҚдёҖдёӘйқһж•°еӯ—еӯ—з¬ҰгҖӮзӯүд»·дәҺ [^0-9]гҖӮ
  • \f: еҢ№й…ҚдёҖдёӘжҚўйЎөз¬ҰгҖӮзӯүд»·дәҺ \x0c е’Ң \cLгҖӮ
  • \n: еҢ№й…ҚдёҖдёӘжҚўиЎҢз¬ҰгҖӮзӯүд»·дәҺ \x0a е’Ң \cJгҖӮ
  • \r: еҢ№й…ҚдёҖдёӘеӣһиҪҰз¬ҰгҖӮзӯүд»·дәҺ \x0d е’Ң \cMгҖӮ
  • \s: еҢ№й…Қд»»дҪ•з©әзҷҪеӯ—з¬ҰпјҢеҢ…жӢ¬з©әж јгҖҒеҲ¶иЎЁз¬ҰгҖҒжҚўйЎөз¬ҰзӯүзӯүгҖӮзӯүд»·дәҺ [\f\n\r\t\v]гҖӮ
  • \S: еҢ№й…Қд»»дҪ•йқһз©әзҷҪеӯ—з¬ҰгҖӮзӯүд»·дәҺ[^\f\n\r\t\v]гҖӮ
  • \t: еҢ№й…ҚдёҖдёӘеҲ¶иЎЁз¬ҰгҖӮзӯүд»·дәҺ \x09 е’Ң \cIгҖӮ
  • \v: еҢ№й…ҚдёҖдёӘеһӮзӣҙеҲ¶иЎЁз¬ҰгҖӮзӯүд»·дәҺ \x0b е’Ң \cKгҖӮ
  • \w: еҢ№й…ҚеҢ…жӢ¬дёӢеҲ’зәҝзҡ„д»»дҪ•еҚ•иҜҚеӯ—з¬ҰгҖӮзӯүд»·дәҺ[A-Za-z0-9_]гҖӮ
  • \W: еҢ№й…Қд»»дҪ•йқһеҚ•иҜҚеӯ—з¬ҰгҖӮзӯүд»·дәҺ [^A-Za-z0-9_]гҖӮ
  • \xn: еҢ№й…Қ nпјҢе…¶дёӯ n дёәеҚҒе…ӯиҝӣеҲ¶иҪ¬д№үеҖјгҖӮеҚҒе…ӯиҝӣеҲ¶иҪ¬д№үеҖјеҝ…йЎ»дёәзЎ®е®ҡзҡ„дёӨдёӘж•°еӯ—й•ҝгҖӮдҫӢеҰӮпјҢ\x41 еҢ№й…Қ AгҖӮ\x041 еҲҷзӯүд»·дәҺ \x04 &1гҖӮжӯЈеҲҷиЎЁиҫҫејҸдёӯеҸҜд»ҘдҪҝз”Ё ASCII зј–з ҒгҖӮ.
